# Statistical Analysis Plan

## Main Test

Use paired tests on matched evaluation examples. Report relative target gain,
retention-adjusted gain, drift difference, confidence intervals, and effect
sizes.

## Multiple Comparisons

Apply Holm correction for target, retention, hallucination drift, layer policy,
and cost claims.

## Seeds And Replicates

Paper-facing results require at least 20 seeds or bootstrap replicates. Lower
counts are diagnostic only.

## Endpoint Baseline Rule

If endpoint-neuron selection is not runnable under the same budget, the main
claim cannot be evaluated.
